# Changelog

All notable changes to CVFoundry-Lite will be documented in this file.

The format is based on Keep a Changelog, and versions follow SemVer.

## [0.3.0] - 2026-01-29

### Added

- `original-setup/` golden copy reset mechanism.
- `build.sh` wrapper for running the build.
- Optional credit/gratitude note and creator link.
- Docs improvements (Mermaid, Jinja templating notes, Windsurf link).

### Changed

- Template layout and styling refinements (accent color, footer build metadata).
